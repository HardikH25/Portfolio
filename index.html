Projects

This portfolio features projects exploring a range of web development concepts. Highlights include a dog gallery built with CSS Grid layout, a moon orbit animation using CSS transitions, animations, and transforms, plus various geometric shapes created with CSS and transform properties. Beyond these, I have developed numerous projects involving HTML forms, tables, and lists. All projects are neatly aligned using display properties like Grid and Flexbox. Additionally, I have applied media queries to ensure responsiveness across different screen sizes. The current webpage itself demonstrates extensive use of Flexbox, Grid, and media queries to deliver a clean, adaptive layout.

Dog Gallery

Created my first dog photo gallery using the CSS Grid layout system, arranging images seamlessly in organized rows and columns. This project allowed me to experiment with multiple grid attributes, making the gallery responsive and visually engaging.

Moon Orbit

Created an animated scene of the moon orbiting the sun entirely with CSS and keyframes, simulating realistic celestial movement. This project allowed me to explore essential CSS animation, transition, and transform attributes.

Profile Card

Made a simple profile card using HTML and CSS with Flexbox for layout. Added media queries to make it responsive on different screen sizes.

Shapes

Created multiple geometric shapes like triangle, diamond, square, and circle entirely with CSS, using properties such as border, transform, and border-radius.

Profile Website

Built a responsive profile website using HTML and CSS, featuring a layout designed with Grid and Flexbox for alignment and structure. The site adapts to different screen sizes using media queries.It also includes an interactive form section for user input, styled with CSS to enhance the overall look.

Ferris Wheel

Made a Ferris wheel animation using CSS animations and transform properties. The rotation effect was created by combining keyframes with transforms to make the wheel and its cabins move smoothly, giving it a realistic spinning motion.
